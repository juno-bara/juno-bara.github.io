The Jekyll theme **juno-barba** v0.1.0 was released!

  

juno-bara is a simple, concise, and powerful Jekyll theme for blogs. It is built on [Poole](https://github.com/poole/poole) and [Pure](https://purecss.io/).

  

For more information about this theme, you can read the [README](https://github.com/vszhub/not-pure-poole/blob/master/README.md) file.
